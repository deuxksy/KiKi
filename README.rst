환경설정
=========
3개의 필수 환경설정 사전에 정의되 있어여만 사용이 가능함

- ZZIZILY_KIKI_MODE
    local,dev,prod 3가지로 나눔 make_config.py 등 패포 및 프로그램 설정에서 사용
- ZZIZILY_KIKI_CRYPTO 암복화 KEY 값
    config.py 에 암복화 처리 또는 소스상에 암복화시
- ZZIZILY_KIKI_HOME
    프로젝트 홈 폴더
